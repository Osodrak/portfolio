## 📂 Projetos

### 1. Visualizador de Ordenação
Animação visual em tempo real de algoritmos de ordenação com barras coloridas.
- **Algoritmos:** Bubble Sort, Selection Sort, Insertion Sort
- **Destaques:** contador de comparações e trocas, velocidade ajustável, log de passos
- **Complexidade:** O(n²) — visualização didática

### 2. To-Do App
Gerenciador de tarefas com arquitetura **MVC em JavaScript puro**.
- **Funcionalidades:** adicionar, concluir, remover tarefas, filtros por status
- **Categorias:** personal, work, study
- **Persistência:** localStorage (dados salvos no navegador)

### 3. Calculadora de Expressões
Parser de expressões matemáticas usando o algoritmo **Shunting-Yard** com **Stack**.
- **Algoritmo:** converte notação infixa → pós-fixa (RPN) e avalia com pilha
- **Destaques:** visualização animada da pilha passo a passo, suporte a parênteses
- **Teclado:** suporte completo via teclado físico

### 4. Quiz de Algoritmos
Quiz interativo sobre **Estruturas de Dados e Big-O Notation**.
- **Questões:** 10 perguntas com explicação detalhada de cada resposta
- **Timer:** 20 segundos por questão
- **Pontuação:** 10 pts por acerto + bônus de velocidade (até +3 pts)
