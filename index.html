02 — Fundamentos

  
Estruturas de Dados

  

    

      LINEAR
      
Array & Listas Ligadas

      
Acesso indexado e inserção dinâmica. Base para listas encadeadas simples e duplamente encadeadas.

      

        Acesso O(1)
        Busca O(n)
        Insert O(1)*
      

    

    

      LIFO
      
Stack (Pilha)

      
Estrutura LIFO com push/pop. Usada em recursão, avaliação de expressões e histórico de ações.

      

        Push O(1)
        Pop O(1)
        Peek O(1)
      

    

    

      FIFO
      
Queue (Fila)

      
Estrutura FIFO com enqueue/dequeue. Essencial para BFS, sistemas de tarefas e buffers de dados.

      

        Enqueue O(1)
        Dequeue O(1)
      

    

    

      HIERÁRQUICO
      
Árvore Binária (BST)

      
Estrutura hierárquica com filhos esquerdo e direito. BST permite busca eficiente e ordenação natural.

      

        Busca O(log n)
        Insert O(log n)
      

    

    

      KEY-VALUE
      
Hash Map / Dicionário

      
Mapeamento chave-valor com acesso em tempo constante. Amplamente usado em caches e índices.

      

        Get O(1)
        Set O(1)
        Delete O(1)
      

    

    

      GRAFOS
      
BFS & DFS

      
Busca em largura e profundidade em grafos. Aplicações em rotas, redes sociais e dependências.

      

        BFS O(V+E)
        DFS O(V+E)

Lógica de Programação

  

    

      
Fibonacci Memoização

      
Reduz complexidade de O(2ⁿ) para O(n) armazenando subproblemas já resolvidos.

      
def fib(n, memo={}):
  if n in memo: return memo[n]
  if n <= 1: return n
  memo[n] = fib(n-1) + fib(n-2)
  return memo[n]

Palíndromo String

      
Verificação com slice invertido do Python. Ignora espaços e diferença de maiúsculas.

      
def is_palindrome(s):
  s = s.lower().replace(' ', '')
  return s == s[::-1]

Binary Search O(log n)

      
Divisão e conquista em array ordenado. Muito mais rápida que busca linear O(n).

      
def binary_search(arr, target):
  lo, hi = 0, len(arr) - 1
  while lo <= hi:
    mid = (lo + hi) // 2
    if arr[mid] == target: return mid
    elif arr[mid] < target: lo = mid+1
    else: hi = mid-1
